# 줄넘기 교내 스포츠클럽대회 현황판 (v19)

학교 줄넘기 대회를 위한 웹 기반 종합 관리 시스템입니다.

## 🎯 주요 기능

### 대회 설정
- 학년/팀, 반, 종목 커스터마이징
- 심판 배정 및 관리
- 16가지 사전정의 종목 지원

### 선수 관리
- 반별 명단 입력 및 종목별 배정
- 드래그&드롭 인터페이스
- 일괄 명단 입력 (텍스트 붙여넣기/파일 업로드)
- 개별 선수 추가/삭제

### 경기 기록
- 팀/개인별 점수 입력
- 직관적인 +/- 버튼 조작
- 실시간 누적점수 계산
- 취소 기능 (최대 10단계)

### 타이머 시스템
- 1분/30초/커스텀 시간 지원
- 전체화면 카운트다운
- 커스텀 음원 업로드
- 시각적 애니메이션 효과

### 결과 발표
- 전체화면 순위 표시
- 팬파레/드럼롤 음향 효과
- 우승팀 강조 표시

### 데이터 관리
- 로컬스토리지 자동 저장
- JSON 파일 백업/복구
- CSV 엑셀 내보내기

## 🚀 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 대회 설정에서 학년, 반, 종목 설정
3. 선수 명단 입력
4. 경기 진행 및 점수 기록
5. 결과 발표

## 💻 기술 스택
